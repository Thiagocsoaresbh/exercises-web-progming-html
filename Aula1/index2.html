Imagens na Web

As imagens são elementos fundamentais para enriquecer o conteúdo visual da página. O elemento utilizado é o , que é um elemento vazio (não tem fechamento).

Atributos importantes de imagens

  • alt: descrição alternativa para acessibilidade.
  • title: texto extra exibido como dica.
  • width/height: controla dimensões.
  • loading: otimiza carregamento.
  • referrerpolicy: controla envio de informações.
